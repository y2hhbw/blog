## welcome to my blog

this is my first blog post using hugo and cloudflare pages!

### features

- fast static site generation with hugo
- free hosting on cloudflare pages
- beautiful papermod theme
- easy to deploy and maintain

### getting started

hugo makes it easy to create and manage blog content. simply create markdown files in the `content/posts` directory and hugo will automatically generate your site.

### why choose this stack?

**hugo** is one of the fastest static site generators available. it builds sites in milliseconds, making development a breeze.

**cloudflare pages** provides:
- global cdn for lightning-fast content delivery
- automatic https with free ssl certificates
- unlimited bandwidth
- git integration for seamless deployments
